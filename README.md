# Keystone - Feature Grid Template

A clean, modern, and responsive feature grid layout designed to showcase key strengths or services in a visually appealing way. Perfect for portfolios, agency websites, or personal branding pages.

## Live Deployment

[View Live Demo](https://thisislefa.github.io/Keystone)

## Features

- **Responsive Design**: Adapts seamlessly across desktop, tablet, and mobile devices
- **Modern UI**: Clean typography, subtle shadows, and structured spacing
- **Feature Grid**: Four visually distinct cards with numbered indicators
- **Interactive Button**: Call-to-action button with arrow icon for engagement
- **Accessible Markup**: Semantic HTML structure for better SEO and accessibility
- **Customizable**: Easy to modify colors, content, and layout

## Project Structure

## Installation & Usage

1. **Clone or download** the project files
2. **Link the CSS** – ensure `style.css` is in the same directory as `index.html`
3. **Customize content** – edit the HTML to reflect your own features and text
4. **Add styling** – create `style.css` to match your design preferences
5. **Deploy** – upload to your web server or hosting platform

## Customization

### Content
- Replace "Why choose me?" with your own heading
- Update feature titles and descriptions in the card sections
- Modify button text and link

### Styling
The template uses:
- **Font**: Inter (loaded from Google Fonts)
- **Colors**: Customizable via CSS variables or direct styling
- **Layout**: Flexbox/Grid-based structure

### Adding More Cards
Duplicate the `.feature-card` article structure and update numbers/content as needed.

## Browser Compatibility

Designed to work with modern browsers that support:
- CSS Grid/Flexbox
- SVG icons
- Custom fonts
